# Figma Code Connect

This repo includes example **Code Connect** mapping files so you can connect Figma components to code.

## Config

A `figma.config.json` file lives at the repo root and tells Code Connect where to find mapping files.

## File format

Figma supports multiple approaches (parser-based and template files). In this sample, we include **React mapping files** alongside each component as:

- `<ComponentName>.figma.tsx`

These are examples you can adapt to your organization’s Code Connect workflow.

## Next steps (outside this sample)

- Install and run the Code Connect CLI to connect/publish mappings.
- Map your published Figma library components to code components via Code Connect UI if desired.
